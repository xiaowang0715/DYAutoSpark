# 抖音自动续火花

![](https://raw.githubusercontent.com/KID-1912/Github-PicGo-Images/master/202403132250853.png)

基于 [AutoJs6 ](https://docs.autojs6.com/#/) 开发的抖音自动化脚本，该程序会定时向抖音好友发送消息已达成续火花的目的，发送后自动强制停止抖音、返回桌面并锁屏，保护隐私且节省电量。

## 功能流程

- 唤醒屏幕

- 解锁手机密码(仅支持数字密码)

- 打开抖音

- 向指定好友发送消息（可自定义消息）

- 完成后台退出 app

- 锁屏 

**注：** 本脚仅供学习，纯属个人娱乐，请勿修改用于其它用途

## 使用说明

### 脚本准备

**下载脚本**

首先将项目 抖音自动续火花程序.js 脚本文件下载后发送到手机，并在AutoJs App 导入


**配置变量**

点击 铅笔编辑图标，修改以下脚本内容

- 修改目标好友名

```js
 var friendNames = ["把此处换成目标好友名"];
```

大概在代码的第19行

注意好友抖音昵称不要填错，需要全称

- 修改锁屏密码（仅限于九宫格数字密码）

```js
 var password = [1,2,3,4,5,6];
```

大概在代码的第20行
 
- 修改好友点击坐标

```
  // 根据设备实际情况换成自己的坐标
    let x = 700;
    let y = 987;
```

大概在代码的第52行

不知道怎么获取好友的点击坐标的看下面

### 获取坐标（以 一加/OPPO 手机为例）

- 点击设置

- 点击关于本机

- 点击版本信息

- 连续点击版本号进入开发者模式

- 回到设置主页面

- 点击系统与更新

- 点击开发者选项

- 下滑找到指针位置并开启

- 打开抖音的“消息”页面

- 把指针滑到你的目标好友

- 屏幕最上方显示的就是指针坐标

- 记录并修改脚本内容

如果是不明白就去网上查阅

### 单次运行

点击三角形启动按钮，尝试单次运行（运行前请后台退出抖音app）
成功单次运行脚本即流程走通，则下一步设置定时运行，否则参考 **常见问题** 部分解决后继续

### 定时运行

- 点击脚本右侧的三个点，找到定时任务

- 将脚本设置成每天定时运行并成功完成后终止



**注：**

好友置顶：为提高成功进入好友聊天框概率，可在抖音 “好友详情” “更多” 开启 “置顶聊天”

建议使用闲置备用机，平时息屏，保证电量及时充电

## 常见问题

### 点击错误页面

如果在脚本运行时进入了错误页面大概率就是因为你的抖音界面停留在了消息页面，你需要做的就是首次执行脚本时先删除抖音后台


### 执行完成之后没有锁屏

在完成消息发送后如果脚本没有使你的手机自动锁屏那么你可以将手机的息屏时间改成10秒（时间越短越好严谨设置成永不息屏）
